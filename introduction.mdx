Figura is a Minecraft Java Edition client mod that replaces the default player model with a fully custom avatar of your own design. Any other player who also has Figura installed will see your avatar automatically — no server-side plugin or mod is required. Avatars are uploaded to the Figura backend server when you equip them and downloaded by other clients on demand.

## How Figura works

Figura is a **client-only** mod. When you equip an avatar, Figura authenticates your Mojang account and uploads your avatar data to the Figura backend server. Other players running Figura connect to the same backend and automatically fetch and render your avatar in-game. The process is transparent — you do not need to configure anything on the Minecraft server you play on.

<Note>
  Figura requires a valid, online Mojang account. Non-premium (cracked) accounts and offline-mode servers cannot upload or download avatars because Figura relies on Mojang authentication to verify account ownership.
</Note>

## Supported mod loaders

Figura supports all major Minecraft Java Edition mod loaders:

- **Fabric** — recommended; widest ecosystem compatibility
- **Forge** — full support
- **Quilt** — supported (Fabric builds run on Quilt)
- **NeoForge** — supported

The current build targets **Minecraft 1.20.1**. Check the download pages for the latest supported versions.
