---
title: "ClientAPI — Minecraft client and game state info"
description: "The client global provides read-only access to Minecraft client state: FPS, version, window size, camera, text dimensions, server info, memory, and more."
---

The `client` global gives your script read-only access to the Minecraft client's current state. All methods on this API are static — they can be called directly on `client` without the colon syntax, though both styles work. Use this API to query FPS, system time, window dimensions, camera position, server information, memory usage, loaded mods, and many other runtime values.

---

## getFPS

Returns the current rendered frames per second as an integer parsed from Minecraft's internal FPS string.

<ResponseField name="return" type="integer">
  Current FPS. Returns `0` if the FPS string is empty.
</ResponseField>

```lua
print("FPS: " .. client.getFPS())
```

## getWindowSize

Returns the window dimensions in pixels. Pass `true` to get the raw framebuffer size; pass `false` or nothing to get the GUI-scaled screen size.

<ParamField path="body.originalSize" type="boolean">
  When `true`, returns the raw framebuffer dimensions. When `false` or `nil`, returns the scaled screen size.
</ParamField>

<ResponseField name="return" type="Vector2">
  Window width and height as a `Vector2`.
</ResponseField>

```lua
local size = client.getWindowSize(false)
print(size.x, size.y)
```

## getTextDimensions

Returns both the width and height of a text string as a `Vector2`. Supports wrapping within a max pixel width.

<ParamField path="body.text" type="string" required>
  The text string to measure.
</ParamField>

<ParamField path="body.maxWidth" type="integer">
  Maximum pixel width before wrapping. Defaults to `0` (no wrap limit).
</ParamField>

<ParamField path="body.wrap" type="boolean">
  Whether to wrap text at `maxWidth`. Defaults to `true`.
</ParamField>

<ParamField path="body.lineSpacing" type="integer">
  Line spacing multiplier. Defaults to `1`.
</ParamField>

<ResponseField name="return" type="Vector2">
  Text dimensions as a `Vector2` (width, height) in pixels.
</ResponseField>

```lua
local dims = client.getTextDimensions("Hello!", 100, true, 1)
print(dims.x, dims.y)
```

## compareVersions

Compares two semantic version strings. Returns a negative number if `version1 < version2`, zero if equal, or a positive number if `version1 > version2`.

<ParamField path="body.version1" type="string" required>
  First version string, e.g. `"1.2.0"`.
</ParamField>

<ParamField path="body.version2" type="string" required>
  Second version string, e.g. `"1.3.0"`.
</ParamField>

<ResponseField name="return" type="integer">
  Negative, zero, or positive comparison result.
</ResponseField>

```lua
local cmp = client.compareVersions("1.0.0", "1.1.0")
if cmp < 0 then
  print("version1 is older")
end
```
